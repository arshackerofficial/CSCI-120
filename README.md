**Course Description:**

This course is an introduction to computing science and program design, suitable for students with little or no programming background. Students will learn fundamental concepts and terminology of computing science, acquire introductory skills for programming in a high-level language, and be introduced to the diverse fields and applications of computing science.

**Course Learning Outcomes:**

Upon successful completion of this course, the student will be able to:

- Learn fundamental concepts and terminology of computing science, acquire elementary skills for programming in a high-level language, and be exposed to diverse fields within, and applications of computing science.
- Recognize compile-time and run-time errors.
- Describe an algorithm with pseudocode.
- Understand how information is represented in the computer.
- Convert integers to binary and vice versa.
- Define and use variables and constants.
- Appreciate the importance of comments and good code layout.
- Write arithmetic expressions and assignment statements.
- Create programs that read and process inputs and display the results.
- Learn how to use and process Python strings.
- Implement decisions using if statements.
- Develop strategies for testing programs.
- Validate user input.
- Implement while and for loops.
- Understand nested loops.
- Implement functions.
- Become familiar with the concepts of parameter passing.
- Develop strategies for decomposing complex tasks into simpler ones.
- Determine the scope of a variable.
- Create simple graphics programs.
- Collect elements using lists and tuples.
- Use the for loop for traversing lists, tuples, and strings.
- Use lists, tuples, and strings with functions.
- Work with a dictionary container.
- Work with a dictionary for table lookups.
- Build and use a set container.
- Learn common set operations for processing data.
- Read and write text files.
- Raise and handle exceptions.
- Learn how to think recursively.
- Understand the relationship between recursion and iteration.
- Analyze programs that are much easier to solve by recursion than by iteration.
- Study several sorting and searching algorithms.
- Understand big-Oh notation.
- Estimate and compare the performance of algorithms.
- Understand the concepts of classes, objects, and encapsulation.
- Implement instance variables, methods, and constructors.
- Design, implement, and test your own classes.
- Understand the behavior of object references.

**Course Content/Schedule:**

Week 1: Introduction to the Course, Introduction to Algorithms and Python, Structural Diagram of the CPU and Primary Memory, Intro to Computing Science and Algorithms, Intro to Python, Python Data Processing

Objectives:

- Learn about computers and programming
- Describe an algorithm with pseudocode
- Understand how a computer represents information
- Recognize compile-time and run-time errors

Week 2: Binary Number System, Selection Constructs, Binary Representation, Conditional Statements in Python

Objectives:

- Understand how to convert decimals to binary and vice versa
- Define and use variables and constants
- Appreciate the importance of good code documentation
- Write arithmetic expressions and assignment statements
- Create programs that can accept and display data
- Implement decision statements using IF

Week 3: General Programming Techniques, Looping Constructs Part 1, Looping Constructs in Python Lab 1

Objectives:

- Develop strategies for testing programs
- Validate user input
- Implement while and for loops in Python

Week 4: Looping Constructs Part 2, Functions Part 1, Loops Continued, Functions Assignment 1

Objectives:

- Understand nested loops
- Implement functions
- Become familiar with the concepts of parameter passing

Week 5: Functions Part 2, Stepwise Refinement, Python Graphical Interface, Functions Continued, Python Turtle Lab 2

Objectives:

- Develop strategies for decomposing complex tasks into simpler ones
- Determine the scope of a variable
- Create simple graphics programs

Week 6: Midterm

Week 7: Data Structures, Lists, Strings, Assignment 2

Objectives:

- Collect elements using lists
- Use the for loop for traversing lists and strings
- Use lists and strings with functions
- Utilize two-dimensional lists

Week 8: Data Structures, Tuples, Dictionaries, Lab 3

\*\*Objectives:

- Know how to create, traverse, and utilize tuples
- Work with a dictionary container
- Work with a dictionary for table lookups

Week 9: Data Structures, Sets, Queues, and Stacks, Lab 4

Objectives:

- Know how to create, traverse, and implement sets, queues, and stacks in Python

Week 10: File Input and Output, Exception Handling, Learn and Practice How to Read and Write Text Files, Python Exceptions Handling, Assignment 3

Objectives:

- Read and write text files
- Raise and handle exceptions

Week 11: Midterm

Week 12: Recursion, Sorting and Searching Algorithms, Big O Notation, Recursion vs Iteration, Sort and Searching Algorithms Lab 4

Objectives:

- Understand what recursion is and when to use recursion
- Differentiate recursion with iteration
- Explore different sort and searching algorithms
- Introduce code complexity using Big O notation

Week 13: Python Object-Oriented Programming 1, Python Classes, Instance Variables and Methods, Instantiating Objects

Objectives:

- Learn object-oriented principles
- Learn how to create classes, instance variables, and methods in Python
- Learn to solve a problem utilizing OOP techniques

Week 14: Final Exam
